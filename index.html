  • FOOD WEBSHOP

    Project description

    It is an ongoing project for a food webshop, that was mainly inspired by webpages like bilkatogo.dk.

    The frontend was initially made with vanilla JavaScript/jQuery and later changed to React.js for better performance.

    The styles are written in pure CSS and the layout follows the mobile-first approach. Desktop styles are still being refined. It has a dummy content that has been generated from wikipedia.

    Currently functioning: responsive page resize and menu; navigation between pages; item search; auto-completing search input; adding item to cart; showing sum of items; emptying cart.
