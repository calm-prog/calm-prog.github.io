  • FOOD WEBSHOP

    Project description

    This is a website designed for local stores to sell their niche products in a common online space.

    The frontend is built with React and Material-UI and Firebase serves as a backend.

    The visitors can search for their desired items, read product details on the product description page, view their cart, and place an order which is then followed by a confirmation message

    Skills gained in this project: reusable logic with React Hooks, React Router, fetch API, client-side form validation
